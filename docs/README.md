# OctoAcme Project Management Documentation

Welcome to the OctoAcme project management documentation hub. This README provides a brief overview of our methodology and quick access to all core process documents. Use these docs as your source of truth for project management at OctoAcme—covering how we start, plan, execute, release, and continuously improve projects.

## Project Management Processes Overview

- **Customer-first:** Prioritize value and usability in every decision.
- **Iterative delivery:** Deliver projects in testable increments to enable continuous improvement and faster feedback.
- **Clear roles:** Defines responsibilities for Project Managers (PM), Product Managers (PdM), Developers, QA, and Stakeholders.
- **Data-informed decisions:** Use measurable outcomes and transparent communication.
- **Psychological safety:** Foster open feedback and learning.
- **Lifecycle:** Initiation → Planning → Execution → Release → Retrospective/Close.
